Privacy Policy - QT DLR

Effective date: 30 April 2026

QT DLR is an offline, single-user utility app for local operational record keeping. This Privacy Policy explains what data the app handles and how it is stored.

1. Data Collection and Use

QT DLR does not require account signup, cloud login, or online profile creation in the offline single-user release. The app stores operational entries created by the user, such as substation setup, daily log records, maintenance records, history, battery/fault/ attendance/handover entries, and report metadata.

2. Storage Location

All data is stored locally on the device using on-device storage only. Data remains on the device unless the user explicitly exports a backup file.

3. Cloud, Tracking, and Third Parties

  • No cloud synchronization is used in this offline release.
  • No analytics SDK is used for user tracking.
  • No advertising SDK is used.
  • No personal data is sold or shared with third parties.

4. Backup Export and Import

The app provides optional local backup export/import features controlled by the user. Exported files are created only when the user triggers export and can be shared by the user through device share options.

5. Data Retention and Deletion

Data is retained locally until the user deletes app data, uses in-app reset options, or uninstalls the app. Users have full control over local data lifecycle.

6. Children

This app is intended for utility/workflow operations and is not directed to children.

7. Security

Reasonable measures are used to store data locally on-device. Device-level security (screen lock, OS protection, and secure sharing practices) remains the user's responsibility.

8. Changes to This Policy

This policy may be updated to reflect app changes or legal requirements. Updated versions will be published at the same policy URL.
